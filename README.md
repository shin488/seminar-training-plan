## 研修の目的

| 分野 | 内容 |
|------|------|
| AI | 人工知能技術の実践的活用 |

---

## AI（人工知能）

### AIによる画像認識・外観検査
- カメラで製品の傷や欠陥を自動検出
- 人の目による検査との違いを学べる

### AIによる異常検知
- 機械の振動や温度の変化から故障の兆候を検出

### AIを活用した需要予測・在庫管理
- 過去のデータを分析し、生産量や在庫を最適化する仕組み

### ディープラーニングの製造業への応用
- 品質管理や検査工程での活用事例

## 観光

### 牧ヶ谷湧水公園（徒歩約5〜10分）
- 牧ヶ谷地域の自然を感じられる公園
- 湧水や緑を楽しむことができる
- 静かな環境で散策や休憩に適している
- AI・IoT施設見学後のリフレッシュ場所としておすすめ

### 光鏡院（徒歩約5分）
- 牧ヶ谷地域にある寺院
- 地域の歴史や文化を感じられる場所
- 落ち着いた雰囲気で短時間の散策に向いている

### 千手寺（徒歩約5分）
- 牧ヶ谷周辺の寺院
- 静かな境内を散策できる
- 地域に根付いた文化や歴史を知ることができる
